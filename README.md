# SPS Malta Excel Add-In

Excel VSTO Add-In developed using C# to validate and process parish record data.

## Features
- Given Name, Surname, Place validation
- Gender & Marital Status checks
- Date validations (Birth–Baptism, Death–Burial, Marriage–Banns)
- Excel merge & split
- Header template validation
- Trim, unmerge, cleanup utility

## Technologies
- C#
- VSTO (Excel Add-In)
- Microsoft Office Interop Excel
- Windows Forms
- .NET Framework

## Usage
Install the Add-In and use ribbon buttons to validate Excel sheets.
